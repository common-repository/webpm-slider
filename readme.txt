Wordpress Image Slider

== Installation ==

1. Upload `webpm-slider` to the `/wp-content/plugins/` directory
   or upload through plugin manager by searching for WebPM Slider
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert the following code.
`<?php if (function_exists('webpm_slider')) {
	webpm_slider();
} ?>`
